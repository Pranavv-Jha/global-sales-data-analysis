# Global Sales Data Analysis

## Overview
This project analyzes a large global sales dataset (1M+ rows) to identify revenue trends, regional performance, and product profitability. The analysis was performed using Python and Pandas to transform raw sales data into meaningful insights.

## Tools Used
- Python
- Pandas
- Data Analysis
- Data Visualization

## Key Analysis
- Cleaned and prepared a large transactional dataset for analysis
- Identified top-performing regions and product categories
- Calculated profit margins and revenue distribution
- Explored sales trends across different markets

## Dataset
The original dataset used for analysis contains more than **1 million rows**.  
A smaller sample dataset has been uploaded in this repository due to GitHub file size limitations.

## Files
- `sales_analysis.py` – Python script used for analysis
- `sales_sample.csv` – Sample dataset
- `README.md` – Project documentation
